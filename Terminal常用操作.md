### Terminal 常用操作命令

1. node 环境执行 JS 文件：`node 要执行的js文件的路径`

2. 切换所处目录 `cd`。可以直接鼠标左键单机目标地址栏，复制路径，然后 `cd 路径`。

   此方法不能切换不同的盘，如当前在 C 盘，直接切换至 D 盘的一个地址，是不可以的。如果要切换至 D 盘，应先输入 D: 切换至 D 盘目录，再输 `cd 路径`。或者直接进入目标目录中输入 cmd。

3. powershell 是 cmd 的升级版，我们用哪个都可以，但是 powershell 好一点。

4. 终端中的快捷键

   1. 上箭头，回到上一次执行的命令。

   2. tab 键，能够快速补全路径。例如目录下有一个js文件：prac1.js。只需要输入 `node p` 然后按 tab，就可以补全路径和文件名。

   3. esc 键，能够快速清空当前已输入的命令。

   4. 输入 `cls` 命令，可以清空终端。
